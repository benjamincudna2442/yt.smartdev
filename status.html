Welcome to SmartDev's API

Seamlessly extract YouTube video and audio formats with detailed metadata and direct download links. Designed for speed, reliability, and ease of use with cookie-based authentication support.

/dl - Fetch Formats

Retrieve all available YouTube video and audio formats with direct download links.

Method: GET

Parameters:

  • url: YouTube video URL (required)
  • format: Desired format ID (optional)
  • quality: Desired quality (optional)
  • type: Media type: video or audio (optional)

Example Request:

curl "https://yt-smartdev.vercel.app//dl?url=https://youtu.be/bUUCZpPhNZs"

Notes: Requires cookies.txt in server directory for authentication.

/docs - API Docs

Access comprehensive API documentation in JSON format.

Method: GET

Example Request:

curl "https://yt-smartdev.vercel.app//docs"

Response: JSON object with endpoint details and API info.
